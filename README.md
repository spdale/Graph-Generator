# Graph-Generator
Generate a graph for a .csv file.

[Web Demo][1]

Basic Startup:
``` 
npm install
npm start
```

Generate Windows .exe:
```
npm install
npm start
npm run package:win
```


Useful Shortcuts:
Reveal in File Explorer: ``` Alt + Shift + R + (Click) ```



[1]:https://spdale.github.io/Graph-Generator/
